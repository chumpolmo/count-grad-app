การติดตั้งซอฟต์แวร์
--
.
********************************************************************
ผู้พัฒนาจำเป็นต้องติดตั้งซอฟต์แวร์และทรัพยากร ดังนี้
- ชุดเครื่องมือ Xampp (PHP, Apache Web Server และ MySQL*)
(*หรือติดตั้ง MySQL Database แยกจากชุดเครื่องมือ Xampp ได้เช่นเดียวกัน)
- MySQL Workbench เพื่อเป็นเครื่องมือสำหรับบริหารจัดการฐานข้อมูล MySQL
********************************************************************
.
1. นำเข้าฐานข้อมูลภายใต้โฟลเดอร์ databases ทุกตารางภายในฐานข้อมูลระบุ
2. กำหนดข้อมูลการเชื่อมต่อฐานข้อมูลในไฟล์ ./config/dbConnection.php
3. กำหนดข้อมูลตั้งต้นของระบบในไฟล์ ./config/settings.php
4. ทดสอบการใช้งานระบบภายใต้โดเมนที่ติดตั้ง ตัวอย่าง http://localhost/count-grad-app
.
