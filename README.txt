การติดตั้งซอฟต์แวร์
--
.
********************************************************************
ผู้พัฒนาจำเป็นต้องติดตั้งซอฟต์แวร์และทรัพยากร ดังนี้
- ชุดเครื่องมือ Xampp (PHP, Apache Web Server, MySQL*, และ PHPMyAdmin**)
(*หรือติดตั้ง MySQL Database แยกจากชุดเครื่องมือ Xampp ได้เช่นเดียวกัน)
(**MySQL Workbench เพื่อเป็นเครื่องมือสำหรับบริหารจัดการฐานข้อมูล MySQL ได้เช่นเดียวกัน)
********************************************************************
.
1. นำโครงสร้างระบบไปวางภายใต้โฟลเดอร์ htdocs
2. นำเข้าฐานข้อมูลภายใต้โฟลเดอร์ databases ทุกตารางภายในฐานข้อมูลระบุ
3. กำหนดข้อมูลการเชื่อมต่อฐานข้อมูลในไฟล์ ./config/dbConnection.php
4. กำหนดข้อมูลตั้งต้นของระบบในไฟล์ ./config/settings.php
5. ทดสอบการใช้งานระบบภายใต้โดเมนที่ติดตั้ง ตัวอย่าง http://localhost/count-grad-app
.
